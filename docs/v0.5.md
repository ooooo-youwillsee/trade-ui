# v0.5

## Summary

`v0.5` 主要强化网格展示、修复合约网格止盈后的未平仓持仓口径，并新增独立的对冲合约网格策略。这个版本同时整理了合约持仓聚合逻辑，删除合约旧兼容结果字段 `floatingProfitLoss`，让合约侧收益字段统一使用已实现收益、未实现收益和总收益。

## Key Features

- 网格展示增强：
  - 网格价格展示相对当前价的涨跌变化，便于快速判断挂单位置。
  - 价格区间展示相对入场价的变化和区间振幅。
  - 网格明细新增单格收益金额，不只展示收益率。
  - 格式化工具补充价格参考变化展示，并增加测试覆盖。
- 合约网格止盈持仓修复：
  - 合约网格计算区分已成交网格、未平仓网格和已止盈平仓网格。
  - 已止盈网格不再计入当前持仓、持仓均价、名义仓位和强平价。
  - 收益拆分为 `realizedProfitLoss`、`unrealizedProfitLoss`、`totalProfitLoss`。
  - 当前权益改为基于未平仓占用保证金和总收益计算。
- 对冲合约网格：
  - 新增独立策略类型“对冲合约网格”，包含多头腿和空头腿两组独立合约网格参数。
  - 支持两条腿分别设置场景涨跌幅，用于估算场景价下的保证金压力。
  - 结果展示当前/场景持仓、当前/场景持仓均价、当前/场景强平价。
  - 汇总展示可转出盈利、需补保证金、资金缺口和场景总收益。
  - 新增独立本地存储、策略列表入口、编辑页、详情页和测试。
- 对冲场景计算：
  - 顺势场景从当前未平仓持仓推演，到达目标价的网格转为已实现收益。
  - 逆势场景继续复用公共合约网格计算，允许新增被触发的网格仓位。
  - 补保证金基于修正后的场景未平仓持仓计算。
- 代码结构重构：
  - 新增公共合约持仓聚合模块 `position.js`，复用持仓聚合和强平价计算。
  - 普通合约网格和对冲合约网格共用持仓聚合口径。
  - 对冲场景计算拆分为顺势、逆势和平仓处理函数。
  - 抽取 `ContractHedgeGridLegFields.vue`，复用多头腿和空头腿表单字段。
  - 对冲网格相关代码补充中文注释。
  - 合约结果删除旧兼容字段 `floatingProfitLoss`，合约侧统一使用 `totalProfitLoss`、`unrealizedProfitLoss` 和 `scenarioTotalProfitLoss`。
