# v0.1

## Summary

`v0.1` 是交易策略工具的初始版本，核心围绕“合约网格”策略的本地计算、保存和展示。该版本提供移动端优先的基础页面结构，包含合约策略列表、合约网格新增/编辑/详情、股票占位页和我的页面。

## Key Features

- 合约网格策略：
  - 支持设置策略名称、价格区间、入场价、当前价、网格模式、网格数量、方向、杠杆、初始保证金和追加保证金。
  - 支持等差网格和等比网格。
  - 支持做多和做空方向。
  - 支持创建时是否包含已开仓网格。
- 合约网格计算：
  - 计算网格价格、已触发网格、持仓数量、持仓均价、浮动盈亏、当前权益。
  - 计算单格收益率、区间收益率、预估强平价和当前持仓强平价。
  - 在详情页展示参数、核心指标、仓位与风险信息、网格明细。
- 本地策略管理：
  - 使用 localStorage 保存合约网格策略。
  - 支持新增、编辑、复制、重置和删除策略。
  - 列表页展示策略卡片和关键指标。
- 应用结构：
  - 底部导航包含合约、股票、我的。
  - 股票页为占位入口。
  - 我的页展示基础信息入口。

## Routes

- `/#/contract`：合约网格策略列表。
- `/#/contract/new`：新增合约网格策略。
- `/#/contract/:id`：合约网格策略详情。
- `/#/contract/:id/edit`：编辑合约网格策略。
- `/#/stock`：股票占位页。
- `/#/mine`：我的页面。

## Storage

- 合约网格策略使用本地存储保存。
- 数据仅保存在当前浏览器环境，不接入交易所 API。

## Assumptions

- 本版本只实现合约网格，不包含现货网格和马丁策略。
- 合约强平价为简化估算结果，不绑定具体交易所的保证金阶梯。
- 股票功能仅作为导航占位，暂不提供策略计算。
