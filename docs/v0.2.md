# v0.2

## Summary

`v0.2` 将策略工具从单一“合约网格”扩展为按市场和策略类型拆分的结构。底部导航调整为合约、现货、股票、我的，并新增合约马丁、现货网格、现货马丁三类策略能力。每类策略拥有独立入口、编辑页、详情页、表单组件、结果组件和本地存储。

## Key Features

- 导航与市场入口：
  - 底部导航拆分为合约、现货、股票、我的。
  - 合约页支持新增合约网格和合约马丁。
  - 现货页支持新增现货网格和现货马丁。
  - 股票页保持占位入口。
- 四类策略线：
  - 合约网格：保留原合约网格计算口径和风险展示。
  - 合约马丁：支持首单金额、加仓倍数、触发涨跌幅、止盈百分比、资金上限、杠杆和保证金风险估算。
  - 现货网格：新增现货网格计算，不展示杠杆、保证金和强平结果。
  - 现货马丁：复用马丁纯计算核心，固定为现货模式，不展示合约风险字段。
- UI 拆分：
  - 每类策略都有独立 EditPage 和 DetailPage。
  - 每类策略都有独立 Form 和 Results 组件。
  - 马丁策略 UI 与网格策略保持一致的编辑页、详情页、参数表单和结果展示结构。
- 本地存储：
  - 合约网格、合约马丁、现货网格、现货马丁分别使用独立 localStorage key。
  - 不迁移旧策略数据，新结构按独立策略线保存。
- 命名规范：
  - 页面统一使用 `*Page.vue` 后缀。
  - 可复用组件使用 `*Form.vue`、`*Results.vue`、`*Panel.vue`、`*Placeholder.vue` 等职责后缀。
  - 合约网格默认参数从 `strategyDefaults` 重命名为 `contractGridDefaults`。

## Routes

- `/#/contract`：合约策略列表。
- `/#/contract/grid/new`：新增合约网格。
- `/#/contract/grid/:id`：合约网格详情。
- `/#/contract/grid/:id/edit`：编辑合约网格。
- `/#/contract/martingale/new`：新增合约马丁。
- `/#/contract/martingale/:id`：合约马丁详情。
- `/#/contract/martingale/:id/edit`：编辑合约马丁。
- `/#/spot`：现货策略列表。
- `/#/spot/grid/new`：新增现货网格。
- `/#/spot/grid/:id`：现货网格详情。
- `/#/spot/grid/:id/edit`：编辑现货网格。
- `/#/spot/martingale/new`：新增现货马丁。
- `/#/spot/martingale/:id`：现货马丁详情。
- `/#/spot/martingale/:id/edit`：编辑现货马丁。
- `/#/stock`：股票占位页。
- `/#/mine`：我的页面。

## Calculation Notes

- 合约网格继续使用原计算逻辑，包含保证金、名义仓位、强平价、权益和风险指标。
- 现货网格只计算实际投入、持仓数量、均价、浮盈亏、单格收益率、区间收益率和当前权益。
- 马丁策略按首单金额、加仓倍数和触发比例生成层级订单。
- 马丁止盈使用整体持仓均价百分比计算。
- 合约马丁强平价仍为简化估算模型，后续可按交易所维护保证金阶梯继续增强。

- 旧本地策略数据不迁移到新结构。
